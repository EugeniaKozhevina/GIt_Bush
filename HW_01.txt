1. Посмотреть где я
	pwd

2. Создать папку
	mkdir foldername_1

3. Зайти в папку
	cd foldername_1

4. Создать 3 папки
	mkdir fn_1 fn_2 fn_3

5. Зайти в любую папку
	cd fn_1

6. Создать 5 файлов (3 txt, 2 json)
	touch f1.txt f2.txt f3.txt f4.json f5.json

7. Создать 3 папки
	mkdir fn_4 fn_5 fn_6

8. Вывести список содержимого папки
	ls
	ls -l (структурированные столбцы, включая права доступа, имя пользователя, к какой группе принадлежит пользователь, размеры файлов, дата создания)
	ls -a (с указанием пути к файлам)
	ls -la (объединяет -l и -a)

9. + Открыть любой txt файл
	cat >> f1.txt

10. + написать туда что-нибудь, любой текст
	так и писать в консоли текст
	перейти на новую строку (Enter)

11. + сохранить и выйти
	Ctrl+C

Альтернативный вариант для 9-11:
	vim f1.txt
	i  и написать текст
	перейти на новую строку (Enter)
	Esc  и  :wq

12. Выйти из папки на уровень выше
	cd ..

13. переместить любые 2 файла, которые вы создали, в любую другую папку
	mv fn_1/f1.txt fn_2/f1.txt | mv fn_1/f5.json fn_3/f5.json

14. скопировать любые 2 файла, которые вы создали, в любую другую папку
	cp fn_2/f1.txt fn_1/f1.txt && cp fn_1/f4.json fn_3/f4.json

15. Найти файл по имени
	find . -name "f1.txt"

16. просмотреть содержимое в реальном времени
	tail -f f1.txt

17. вывести несколько первых строк из текстового файла
	head f1.txt

18. вывести несколько последних строк из текстового файла
	tail -f f1.txt

19. просмотреть содержимое длинного файла
	less f1.txt

20. вывести дату и время
	date +%c
